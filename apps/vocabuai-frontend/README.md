# VocabuAI Frontend

React Native + Expo frontend for the VocabuAI application with React Native Web support for cross-platform development (Android, iOS, and Web).

## Getting Started

### Prerequisites
- Node.js 18+
- npm
- For Android: Android Studio and Android SDK
- For iOS: macOS with Xcode (optional, can use Expo Go app)

### Installation
```bash
npm install
```

### Development

#### Start the development server
```bash
npm start
```

#### Run on Android
```bash
npm run android
```

#### Run on iOS
```bash
npm run ios
```

#### Run on Web
```bash
npm run web
```

## Technologies

- **React Native**: Cross-platform mobile framework
- **Expo**: Development platform for React Native
- **React Native Web**: Run React Native components on the web
- **TypeScript**: Type-safe development

## Features

- Cross-platform support (Android, iOS, Web)
- Modern React Native with the new architecture enabled
- TypeScript for type safety
- Expo managed workflow for easier development
